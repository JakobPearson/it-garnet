Thoughts on Progressive Enhancement-

Progressive Enhancement is a method of coding that prioritizes its process into specified steps to developing a website.More so however, it is a method that focuses heavily on the content of a site and making the basic content accessible for older and newer devices alike. First is the content, always start with content when using Progressiv Enhancement. This way, you can model your website around the content within it with HTML. Next you would use CSS to style the site and make it look how you'd like. Finally add javascript for the responsive aspects of your site for user interaction. This is important in modern web development because code languages and devices are improving at a rapid rate and it is important to make sure that web developers prioritize accessibilty of content above all else. If not for the content a website would be pointless.
